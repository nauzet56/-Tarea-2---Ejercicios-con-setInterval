# -Tarea-2---Ejercicios-con-setInterval# Tarea de JavaScript - Nauzet

Aquí dejo los ejercicios de la tarea de JavaScript. Todos los archivos están en la raíz del repositorio.

## Ejercicios:

* **ejer1.html**: Un reloj en tiempo real que se actualiza cada segundo solo.
* **ejer2.html**: Cuenta atrás del 10 al 0 con botones de iniciar y parar. Al llegar al final pone "¡Tiempo!".
* **ejer3.html**: Un párrafo que parpadea cada 500ms usando un botón para empezar y otro para frenarlo.
* **ejer4.html**: Una barra de progreso que se llena en 10 segundos y avisa cuando llega al 100%.
* **ejer5.html**: El juego de parar el número aleatorio. Hay que intentar frenarlo en el 7 para ganar.
